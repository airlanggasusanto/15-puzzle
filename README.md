# 15 Puzzle (Unity)

A Unity implementation of the classic 15-puzzle game built with Unity UI Toolkit.

This project recreates the sliding puzzle experience with a desktop-focused UI, keyboard navigation, animations, pause system, and solvable board generation.

---

## Features

- Classic 4x4 sliding puzzle gameplay
- Solvable randomized board generation
- Keyboard controls (arrow keys)
- Mouse click interaction
- Move counter tracking
- Timer system
- Pause menu (ESC / Space)
- Win detection
- Tile animations
- Desktop UI layout

---

## Controls

- Arrow Keys → Move tiles
- Mouse Click → Move tiles
- ESC → Pause / Resume
- Space → Pause / Resume

---

## Build

Built with:
- Unity UI Toolkit
- C#

To run:
1. Open project in Unity
2. Load main scene
3. Press Play or build for Windows (x86_64)
